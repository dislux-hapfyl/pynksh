# pynksh

shell functions that wrap python syntax

a small tool to help write shorthand "python" in shell


## How to operate

Launch the IDE from terminal: `ksh newide | python3 &`

Inspect from newide: 
  
  `Ctrl-q` View `newide`
  
  `Ctrl-Shift-q` View `pnk.lang`
